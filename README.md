# ReviewIt
This is a simple code review bot

# Venv Setup

```
python -m venv .venv
source .venv/bin/activatepip install Flask PyJWT requests

```

# Requirements
```
pip install -r requirements.txt
```

Creating Requirements: 

```
pip freeze > requirements.txt
```

# Running the flask server

```
flask run --debug 
```

OR 

```
flask shell
```
